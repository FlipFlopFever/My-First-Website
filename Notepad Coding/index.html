Color Attributes

In HTML, a color can be specified by using a color name with "background-color:___":

"Tomato"

"Orange"

"DodgerBlue"

"MediumSeaGreen"

"Gray"

"SlateBlue"

"Violet"

"LightGray"

Or, alternatively, you can use HEX, RGB, RGBA, HSL, or HSLA values:

rgb(255, 99, 71)

#ff6347

hsl(9, 100%, 64%)

rgba(255, 99, 71, 0.5

hsla(9, 100%, 64%, 0.5)

These color attributes can change the background of the text ("background-color:___") and the color of the text ("color:___"). Furthermore, you can even add a colored border around text, as shown below:
